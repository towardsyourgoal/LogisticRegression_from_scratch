# Logistic Regression 
This repository provides details about the implementation of Logistic Regression without using any pre-build libraries such as sklearn.

## There are below steps-
1. Initialize the parameters
2. Train the model using Linear equations and passing it to the Sigmoid function to generate the probabilities for each sample
3. Make predictions with the new data, and get the class labels as the output instead of the probabilities
4. Evaluation the model performance using metrics such as,

- Mean squared error
- Accuracy
- Precision
- Recall
- F1 score
- FPR
- Specificity
- ROC AUC score

Kindly go through the demo file to understand how to use this.
